### find(query, [opts])

#### query

*Required*<br>
Type: `String`

Perform a brand search inside the directory based on the query.

#### opts

#### findType

Type: `Boolean`<br>
Default: `true`

Determinate if perform a search on type as well.

#### strmatchOpts

Type: `Object`<br>

Setup string matcher options, based on [str-match](https://github.com/Kikobeats/str-match).
